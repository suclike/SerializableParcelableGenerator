SerializableParcelableGenerator

SerializableParcelableGeneratorPlugin for IntelliJ IDEA & Android Studio to generator SerializableParcelable code.

Usage

  1. create or open a Java source file with your desired functionality
  2. right click 'Generate' and 'SerializableParcelable'
  3. select field
  4. and generator code

SerializableParcelableGeneratorPlugin is available at JetBrains Plugin Repository and can therefore be found in your IDE's plugin manager! Alternatively, you can download the plugin from the release section or download from Github Repository.
